# CI/CD Testing with GitHub Workflows

A **learning project** to practice **CI/CD workflows** using **React**, **Vite**, and **GitHub Actions**. This project demonstrates automated build, test, and deployment pipelines for a frontend application.  

---



- Setup of **GitHub Actions** workflows for CI/CD  
- Automated testing on push and pull requests  
- Build and lint checks for a React + Vite project  
- Learn deployment and workflow automation
